# LP02 - Implement and Manage Storage in Azure

## Modules (Sequence)

1. **M01 - Storage Accounts Foundations**
2. **M02 - Blob Services and Data Protection**
3. **M03 - Azure Files, Sync, and Data Movement**
4. **M04 - Storage Security and Governance**

## Student expectations

For each module, students must produce:

- Portal lab (Beginner: click-by-click)
- Az CLI lab (copy/paste)
- PowerShell lab (copy/paste)
- Bicep lab (where applicable)
- Validation script (meaningful pass/fail)
- References (official)
