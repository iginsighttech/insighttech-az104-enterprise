# Lab 01 (Intermediate): Cosmos DB Query Patterns

## Scenario
Design and test a Cosmos DB-backed application workflow with partition-aware reads, writes, and query validation.

## Tasks
1. Create Cosmos DB resources for the scenario.
2. Implement sample data writes.
3. Test partition-aware queries.
4. Review throughput and query behavior.

## Validation Checks
- Data inserts succeed
- Queries return expected results
- Partition key usage is documented and defensible
