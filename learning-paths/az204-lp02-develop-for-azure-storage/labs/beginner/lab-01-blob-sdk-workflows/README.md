# Lab 01 (Beginner): Blob SDK Workflows

## Scenario
Implement a simple application workflow that uploads, lists, and retrieves blobs using an Azure Storage SDK and captures validation evidence.

## Tasks
1. Create storage resources for the lab.
2. Implement upload and retrieval logic.
3. Validate successful reads and writes.
4. Record outputs for instructor review.

## Validation Checks
- Blob upload succeeds
- Blob retrieval returns expected content
- Application logs show successful storage operations
