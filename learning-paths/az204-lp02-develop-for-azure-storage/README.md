# AZ-204 LP02: Develop for Azure Storage

## Overview
This learning path covers application-facing storage patterns for AZ-204, with emphasis on Blob Storage, Cosmos DB, SDK usage, and data design decisions.

## Outcomes
- Implement Blob Storage workflows in application scenarios.
- Design and test Cosmos DB access patterns.
- Apply secure connection and identity options for storage access.
- Validate storage behavior with repeatable checks.
