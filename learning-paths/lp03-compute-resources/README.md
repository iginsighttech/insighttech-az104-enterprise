# LP03 - Deploy and Manage Azure Compute Resources

## Modules (Sequence)

1. **M01 - Virtual Machine Foundations**
2. **M02 - VM Scale and Availability**
3. **M03 - App Service and Container Services**
4. **M04 - Compute Security and Operations**

## Student expectations

For each module, students must produce:

- Portal lab (Beginner: click-by-click)
- Az CLI lab (copy/paste)
- PowerShell lab (copy/paste)
- Bicep lab (where applicable)
- Validation script (meaningful pass/fail)
- References (official)

## Practice exam

- `exams/practice-50q.md` (questions scaffold)
- `exams/answer-key.md` (answer key scaffold)
