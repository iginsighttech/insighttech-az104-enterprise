# LP04 - Configure and Manage Virtual Networks for Azure Administrators

## Modules (Sequence)

1. **M01 - Virtual Network Foundations**
2. **M02 - Connectivity and Routing**
3. **M03 - Load Balancing and Name Resolution**
4. **M04 - Network Security and Troubleshooting**

## Student expectations

For each module, students must produce:

- Portal lab (Beginner: click-by-click)
- Az CLI lab (copy/paste)
- PowerShell lab (copy/paste)
- Bicep lab (where applicable)
- Validation script (meaningful pass/fail)
- References (official)
